# Fowan 工具箱 Changelog

## 0.2.0 - 2026-07-23

- 新增独立汇报工具入口，并将其与 AI Chat、AI 配置中心一同纳入 Windows 发布包和发布说明。

## 0.1.4 - 2026-07-13

- 新增开机自启动安装选项和工具箱设置开关，自启动时无感驻留系统托盘。
- 修复升级安装时旧版 Fowan 进程无法退出导致的文件占用。
- 修复安装器更新日志中文乱码。
- 安装或更新完成后默认启动 Fowan 工具箱。

## 0.1.3 - 2026-07-12

- 更新随工具箱发布的 Todo 任务管理器，包含回收站、筛选、任务详情和清单配色功能。
- 加强 Todo 本地数据写入的并发保护与损坏文件恢复能力。

## 0.1.2 - 2026-07-10

- 将独立的 Fowan 日记工具纳入正式发布包，包含本地日记库、日记本、时间线、标签管理及隐私控制。
- 工具箱入口更新为启动完整日记工具，而非仅展示视觉壳。
- 发布包同时包含更新后的 Todo 主窗口与便签模式运行时。

## 0.1.1 - 2026-07-07

- 新增日记工具入口，用于启动独立 Windows 日记视觉壳。
- 新增启动时自动检查 GitHub Release 稳定版更新。
- 新增更新提示弹窗，支持立即更新、忽略当前版本、稍后提醒和禁用自动检查。
- 新增发布用 `fowan-update.json` manifest，用于校验并下载安装包。

## 0.1.0 - 2026-07-06

- 新增 Windows 工具箱壳层，包含工具搜索、分类、详情面板、设置、诊断、用户资料基础能力和托盘行为。
- 新增组件版本元数据，让打包版本可以一致展示工具箱版本。
- 新增安装器更新日志支持，用于升级安装时提示本次版本变化。
