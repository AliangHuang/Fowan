# Fowan Todo Changelog

## 0.1.4 - 2026-07-13

- 随 Fowan 0.1.4 发布包更新；Todo 功能没有用户可见变化。

## 0.1.3 - 2026-07-12

- 新增回收站、保留期限设置及整棵任务树的恢复和永久删除操作。
- 新增按清单、层级和日期范围筛选任务，以及可选的清单配色。
- 增加主窗口和便签模式的任务详情、右键菜单、拖拽自动滚动与悬浮反馈。
- 改进主窗口标题栏命中处理，并增强主窗口与便签进程共享本地数据时的可靠性。

## 0.1.2 - 2026-07-10

- 优化主窗口与便签模式的单实例激活和预热，减少首次切换延迟。
- 增强任务列表交互，包括已完成任务的展开、恢复、清除及拖放整理。
- 修复主题、启动与窗口恢复相关的稳定性问题。

## 0.1.1 - 2026-07-07

- 随工具箱 0.1.1 打包发布；Todo 功能无用户可见变化。

## 0.1.0 - 2026-07-06

- 新增独立 Windows Todo 工具，使用本地任务存储。
- 新增 Todo 便签模式伴随程序。
- 新增随包工具版本元数据和安装包更新日志覆盖。
