# Fowan Todo Changelog

## 0.2.1 - 2026-07-23

- Todo 与便签改用整套 Fowan 共享运行时交付，缩小安装包而不改变本地任务数据、窗口行为或独立启动方式。

## 0.2.0 - 2026-07-23

- 提取并复用既有 Todo 筛选与日期区间语义，供独立汇报工具生成只读任务快照；Todo 数据格式和现有交互不变。

## 0.1.4 - 2026-07-13

- 新增首次进入主界面的两步聚光新手引导，并可在设置中重新开始引导。
- 新增便签贴边悬浮模式：拖到屏幕左右边缘可收起为 52×52 悬浮图标，支持跨显示器拖动、位置持久化和单击恢复。
- 改进便签窗口操作，新增最小化按钮，并优化拖动、缩放、置顶以及窗口位置与尺寸恢复行为。
- 改进已完成时间编辑器，修复其边框与内容裁切问题，并修复教程、帮助和对话框显示期间虚拟标题栏影响点击的问题。

## 0.1.3 - 2026-07-12

- 新增回收站、保留期限设置及整棵任务树的恢复和永久删除操作。
- 新增按清单、层级和日期范围筛选任务，以及可选的清单配色。
- 增加主窗口和便签模式的任务详情、右键菜单、拖拽自动滚动与悬浮反馈。
- 改进主窗口标题栏命中处理，并增强主窗口与便签进程共享本地数据时的可靠性。

## 0.1.2 - 2026-07-10

- 优化主窗口与便签模式的单实例激活和预热，减少首次切换延迟。
- 增强任务列表交互，包括已完成任务的展开、恢复、清除及拖放整理。
- 修复主题、启动与窗口恢复相关的稳定性问题。

## 0.1.1 - 2026-07-07

- 随工具箱 0.1.1 打包发布；Todo 功能无用户可见变化。

## 0.1.0 - 2026-07-06

- 新增独立 Windows Todo 工具，使用本地任务存储。
- 新增 Todo 便签模式伴随程序。
- 新增随包工具版本元数据和安装包更新日志覆盖。
