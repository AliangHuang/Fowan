# Fowan Diary Changelog

## 0.2.0 - 2026-07-23

- 日记工具升级为可独立使用的本地应用：支持新建、编辑、草稿、收藏、日记本和标签管理，以及日历与时间线筛选。
- 新增图片附件、待办只读关联和 Markdown 导出；地点、天气填充均需在前台获得用户明确授权，默认不会后台定位或刷新。

## 0.1.4 - 2026-07-13

- 随 Fowan 0.1.4 发布包更新；日记功能没有用户可见变化。

## 0.1.3 - 2026-07-12

- 随 Fowan 0.1.3 发布包更新；日记功能没有用户可见变化。

## 0.1.2 - 2026-07-10

- 日记工具从视觉壳升级为本地可用版本：支持新建、编辑、草稿、收藏、删除确认、日记本管理、标签筛选、日历筛选和 Markdown 导出。
- 新增本地 JSON 日记库与损坏数据备份恢复；日记详情可单向关联待办而不修改待办数据。
- 保留三栏日记视觉基准，并让图片附件、标签、模板和日记内搜索在原快速记录工具栏中实际可用。
- 时间线支持按日记本切换和全部日记本总览，并以倒序连续记录展示跨日节点。
- 时间线重构为独立全宽工作区：新增范围筛选、活动日历、时间导航和搜索定位，并移除该页面的快速记录与右侧详情面板。
- 元信息条改为独立的心情、天气、地点下拉；地点支持手动输入、最近地点和用户主动触发的自动定位。
- 新增标签表、标签管理页和 12 色配色；历史字符串标签会兼容迁移并支持筛选、创建、改名和删除定义。
- 新增定位与自动天气的双重显式授权、设置开关和第三方最小化数据传输说明；默认关闭且不会后台定位或刷新天气。

## 0.1.1 - 2026-07-07

- 新增独立 Windows 日记视觉壳，包含三栏主页面、主题设置、示例日记数据和只读待办候选展示。
